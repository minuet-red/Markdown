# git修改提交作者和邮箱

// 设置全局
git config --global user.name "Author Name"
git config --global user.email "Author Email"

// 或者设置本地项目库配置
git config user.name "Author Name"
git config user.email "Author Email"

# 上传json配置和扩展
shift + alt + u
# 另一台电脑下载json配置和扩展
- 下载 setting sync 扩展
- 登录GitHub账号，输入token key和gist ID
- shift + alt + d

# 上传本地代码到GitHub（push）
- 登录GitHub账号
- NEW 新建一个仓库
- 初始化文件夹 + 输入master
- ... -> 拉取、推送 -> 推送到...
- 添加远程（GitHub）
- 输入仓库名称
- 输入取名分支名称
- 点左下角同步键
# 下载GitHub代码到本地文件夹（pull）
- ... -> 拉取、推送 -> 拉取自...
- 输入仓库名称
- 点左下角同步键
